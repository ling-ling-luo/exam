# 命令行视频剪辑

## 题目目标

实现一个带 TUI 的命令行视频剪辑工具，支持多段素材切分、拼接与导出。
- 面对真实输入时可稳定运行
- 交互流程可理解、可操作
- 输出结果可验证、可复现

## 已提供素材

目录：`video_res/`

实际素材以目录内容为准，可能包含不同容器/编码格式，以及异常文件。

## 功能要求（必做）

1. TUI 交互
- 浏览素材列表
- 设置切分区间
- 调整片段顺序
- 执行导出并展示结果

2. 剪辑与合成
- 支持单视频切分
- 支持多视频拼接
- 输出常见可播放格式

3. 错误处理
- 对非法输入给出明确提示
- 不允许静默失败

## 约束条件
- 交互提示应面向用户可理解，而不是仅透传底层报错。

## 验收方式（必做）

请自行设计验证方案，至少覆盖：
- 基础功能正确性
- 异构输入兼容性
- 异常输入鲁棒性
- 连续执行稳定性

## 交付物

- 源码与启动命令（如 `./run.sh`）
- 使用说明（关键操作与参数）
- 示例输入与输出结果
- 验证说明（场景、命令、结果摘要）
- 已知限制与优化方向

## 加分项

- 时间线或片段预览能力
- 任务队列与取消机制
- 可配置导出参数（码率、分辨率、帧率）
